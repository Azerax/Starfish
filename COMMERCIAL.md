# Project Starfish — Commercial & Enterprise

**You do NOT need a commercial license to use Project Starfish — including at a company.** The core
and CLI are Apache-2.0: free for personal *and* commercial use, forever. Adopt it, embed it, ship it.

A separate **commercial offering** exists for the things organizations (not individuals) need, which
are not part of the open core:

- **Compliance & assurance** — external audit-anchoring / notarization, trusted timestamping,
  17a-4 / SOC 2 / SR 11-7 reporting, and an independent auditor node.
- **Scale & control** — SSO / SAML, RBAC, multi-operator federation, fleet management.
- **Marketplace & trust services** — the signed skill registry, publisher identity, and the
  remote-revocation ("kill-switch") service.
- **Managed cloud** — hosted governance + dashboards.
- **Support** — SLAs, onboarding, and **Starfish certification**.

These layers fund continued open development of the core. Using the open core never requires payment.
