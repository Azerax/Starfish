# project-starfish

The `starfish` CLI — a governance-first, deny-by-default overlay that brings any Claude / AI agent &
skill build under governance. See https://projectstarfish.ca and the [repo](https://github.com/Azerax/Starfish).

```bash
npm i -g project-starfish      # or: npx project-starfish govern <pack-dir>
starfish govern ./my-skill-pack
```

Self-contained (no runtime dependencies). MIT.
